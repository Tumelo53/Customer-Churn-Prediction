# Customer-Churn-Prediction

Project Overview

This project implements a supervised machine learning model to predict customer churn using behavioral and subscription data.

The dataset contains over 440,000 customer records including demographic, subscription, and engagement features.

Objective

To identify customers at risk of churning based on historical behavioral patterns and service usage metrics.
